> **Notice**  
> Please note that the module is made by our developer partner, CynderTech. However, this is not to be considered as the official Magento module for PayMongo. Our official plugin is still in the works but feel free to reach out if you have any concerns.
> 
> **Available payment methods**  
> Cards: Visa and Mastercard  
> E-wallet: GCash and GrabPay  
> Buy Now, Pay Later: BillEase
> 
> **Supported versions**  
> The PayMongo Magento module is available for Magento version 2.3 and above. It might not work as intended for versions 2.2 and below - exercise caution and test extensively for Magento 2.2 and older.
> 
> **Supported currency**  
> Your Magento store must be set in Philippine peso (PHP) to use PayMongo. It will not work for other currencies.

If you have a Magento store, you can integrate with PayMongo with no coding required! All you'll need is a PayMongo account.

If you don't have a PayMongo account, you can sign up [here](https://dashboard.paymongo.com/signup) 

### Installing PayMongo on Magento

Select the installation and troubleshooting guides for your preferred installation method below 

* [CLI Installation](https://cynder.atlassian.net/servicedesk/customer/portal/1/article/1017282562)
* [Installation via the configuration manager](https://cynder.atlassian.net/servicedesk/customer/portal/1/article/1108770829)
* [Manual installation](https://cynder.atlassian.net/servicedesk/customer/portal/1/article/551321608)

After installing the module with one of the methods above, configure the plugin in the admin panel of your Magento store. The configuration guide can be found [here](https://cynder.atlassian.net/servicedesk/customer/portal/1/article/1110474759).
